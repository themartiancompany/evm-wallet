Make Seed
---------------------------------------------------------
:Version: mkseed |version|
:Manual section: 1

Synopsis
========

mkseed *[options]* *out_file*

Description
===========
Generates a twelve words seed phrase.

========

-h                      This message.
-c                      Enable color output
-v                      Enable verbose output
